# Kanha Tiger Guide

A complete 28-page SEO/AEO/GEO-optimized static site about Kanha National Park, Madhya Pradesh, India. Built with Astro + Tailwind CSS, deployed to GitHub Pages.

## Site Structure

- **6 Pillar Pages:** Home, Safari Guide, Resorts, Safari Zones, Wildlife, Travel Info
- **22 Sub-Pages:** Detailed content under each pillar
- **All Pages:** Link back to https://kanhasafaristay.com

## Tech Stack

- **Astro** - Static site generator (best for SEO)
- **Tailwind CSS** - Styling
- **MDX** - Rich content support
- **GitHub Pages** - Free hosting
- **GitHub Actions** - Auto-deployment

## Local Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

The site auto-deploys to GitHub Pages on push to `main` branch via `.github/workflows/deploy.yml`.

Site URL: https://Abhishek-parate.github.io/kanha-tiger-guide/

## Content Strategy

Every page includes:
- **SEO:** Proper H1-H6 hierarchy, meta tags, canonical URLs
- **AEO:** Answer-first content, question-based headers, FAQ sections
- **GEO:** 134-167 word citable passages, schema markup, structured data
- **Backlinks:** Contextual links to kanhasafaristay.com specific pages
